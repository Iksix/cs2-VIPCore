# Allows to modify Default Ammo Clip and Reserve for weapons 

# Config

### in vip.json
```json
  "CustomDefaultAmmo": {
    "weapon_ak47": {
      "DefaultClip": 30,
      "DefaultReserve": 120
    },
    "weapon_m4a1": {
      "DefaultClip": 30,
      "DefaultReserve": 120
    },
    "weapon_awp": {
      "DefaultClip": 10,
      "DefaultReserve": 30
    },
    "weapon_deagle": {
      "DefaultClip": 7,
      "DefaultReserve": 35
    }
    // Add more weapons as needed
}
```

# in lang (not necessarily)


EN: 
```json
"CustomDefaultAmmo" : "Custom Default Ammo"
```
